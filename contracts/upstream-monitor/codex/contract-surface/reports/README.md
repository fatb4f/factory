# Codex impact reports

This directory contains non-authoritative projections produced by the scheduled ChatGPT actuator.

- Run reports: `runs/<run_id>.codex-impact.md`
- Current projection: `latest.codex-impact.md`

Report paths and write order are controlled by `contracts/factory/workers/codex/upstream-monitor/publication.cue`. Historical pre-regeneration reports remain available through Git history and are not restored as current state.
